# Basic MYSQL Commands

1. LOGIN TO SQL TARGET
```
mysql -h <TARGET> -u <USER>
```

2. LIST ALL DATABASES:
```
SHOW DATABASES
```

3. SELECT A DATABASE:
```
USE <DATABASE NAME>
```

4. QUERY TO DISPLAY TOATAL NO. OF ENTRIES
```
SELECT COUNT(*) FROM <DB NAME>
```

5. QUERY TO DISPLAY ALL ENTRIES IN A DATABASES
```
SELECT * FROM <DB NAME>
```
